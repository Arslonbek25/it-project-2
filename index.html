Ta'sir etuvchi elementlar(psevdo-elementlar)

CSS psevdo-elementli elementning belilangan qismlarini stillash uchun ishlatiladi.

Masalan, u quyidagilar uchun ishlatilishi mumkin:

  • Elementning birinchi harifni yoki satrini stillash
  • Kontentni element keontentidan oldin yoki keyin kontentni qo'yish

Sintaksis:

selektor::psevdo-element {
    xossa: qiymat;
}

E'tibor bering, ikki nuqtali- ::first-line, ::first-line ni bir shakli.

CSS3 psevdo-elementlar uchun ikki nuqtali notatsiya, bitta nuqtali notatsiyani o'rniga ishlatiladi. Bu W3C tomonidan psevdo-elementlarni farqlash uchun qilingan harakat.

Bitta nuqtali notatsiya CSS2 va CSS1 da ikkala psevdo-element va klass- lar uchun yagona belgilash turi bo'lgan.

::first-line psevdo elementi

::first-line bu matnning birinchi qatoriga allohida still berish uchun ishlatiladi.

<p>Lorem, ipsum dolor sit amet consectetur adipisicing elit.
Magni corrupti, totam suscipit exercitationem eligendi modi eius dignissimos 
facilis neque aspernatur.</p>

p::fist-line {
    color: #f00;
    font-variant: small-caps;
}

Lorem, ipsum dolor sit amet consectetur adipisicing elit. Magni corrupti, totam suscipit exercitationem eligendi modi eius dignissimos facilis neque aspernatur.

::first-line psevdo-elementi faqat blok-elementlar uchun ishlatilishi mumkin.

Quyidagi xossalar ::first-line psevdo-elementi uchun qo'llanadi:

  • font xossalari
  • color xossalari
  • background
  • xossalari
  • word-spacing
  • letter- spacing
  • text-decoration
  • vertical-align
  • text-transform
  • text-height
  • clear

::first-letter psevdo-elementi

::first-line bu matnning birinchi harfiga alohida stil berish uchun ishlatiladi.

p::fist-letter {
